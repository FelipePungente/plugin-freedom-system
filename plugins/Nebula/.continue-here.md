## Architecture Summary

- **Core DSP:** 4 components — juce::dsp::Reverb, Mistero IIR LPF, Pink Noise Generator, DryWetMixer
- **Signal chain:** dry capture → reverb → Mistero LPF → pink noise injection → dry/wet blend
- **Key architectural decision:** Noise injected AFTER Mistero filter (tonally coherent at extreme settings)
- **Risk level:** LOW overall — no FFT, no MIDI, no file I/O, no feedback loops
- **Primary reference implementation:** DriveVerb (identical DSP pattern)
